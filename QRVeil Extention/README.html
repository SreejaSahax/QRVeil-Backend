QRVeil: QR Detector Chrome Extension

QRVeil is a lightweight and sleek Chrome extension that allows you to quickly scan and decode QR codes directly from your browser popup window.

🚀 Features

  • Real-time QR code detection
  • Copy decoded result to clipboard
  • Open scanned URL in a new tab

🛠️ Installation

  1. Download or clone this repository
  2. Go to chrome://extensions/ in Chrome
  3. Enable Developer Mode
  4. Click Load Unpacked and select the extension folder
